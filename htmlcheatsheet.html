HTML/CSS Cheatsheet

<link> Link Element

Link element is used to link HTML documents to external resources like CSS files.

  • href attribute
  • rel attribute
  • type attribute

<link href="./path/to/stylesheet/style.css" rel="stylesheet" type="text/css">
